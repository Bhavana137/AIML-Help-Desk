# 🎓 AIML Help Desk

An AI-powered Academic Help Desk chatbot built using **Python, Flask, LangChain, Hugging Face, and ChromaDB**. The chatbot answers student queries by retrieving information from PDF documents such as syllabus, placement details, hostel information, and academic circulars.

---

## 🚀 Features

- 📚 Syllabus Information
- 💼 Placement Details
- 🏠 Hostel Information
- 📄 Academic Circulars
- 🎓 College Information
- 🤖 AI-powered Question Answering using Retrieval-Augmented Generation (RAG)
- 💬 Interactive Chat Interface
- ⚡ Fast and User-Friendly Design

---

## 🛠️ Technologies Used

- Python
- Flask
- LangChain
- Hugging Face Transformers
- ChromaDB (Vector Database)
- PyPDFLoader
- HTML5
- CSS3 (Internal CSS)
- JavaScript (Internal JavaScript)

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/AIML-Help-Desk.git
```

### 2. Open the Project

```bash
cd AIML-Help-Desk
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
python app.py
```

### 5. Open in Browser

```
http://127.0.0.1:5000/
```
